# 오행 지역 매핑 룰셋 — v1 (지식 기반)

> 250개 시군구 각각에 오행(목·화·토·금·수)을 부여한 **근거와 알고리즘**.
> 구현: [`scripts/gen-elements.mjs`](../scripts/gen-elements.mjs) → 생성물 [`lib/municipalityElements.ts`](../lib/municipalityElements.ts)
> 재생성: `node scripts/gen-elements.mjs`

---

## 0. 왜 이 매핑이 필요한가

"화 기운이 강한 사람에게 동해(수)를 추천한다"가 임의 규칙이면 설득력이 없다.
전통 명리·풍수에는 이미 **방위–오행 체계**와 **지형–오행 체계(형기론)**가 있으므로, 이를 한국 행정구역에 적용해 *근거 있는* 추천을 만든다.

- **목적**: 시군구 → 오행 1개 (지도 선명/흐림 + 추천 로직의 기반)
- **범위**: 전국 250개 시군구 (행정표준 2018 기준, southkorea-maps)
- **성격**: 규칙 기반 결정론적 매핑(v1). 시도당/시군구당 **대표 오행 1개**로 단순화.

---

## 1. 이론 근거

### 1.1 방위–오행 (오방)

| 방위 | 오행 | 오방색 | 비고 |
|---|---|---|---|
| 동(東) | 목(木) | 청(靑) | 상승·성장 |
| 남(南) | 화(火) | 적(赤) | 확산·발산 |
| 중앙(中) | 토(土) | 황(黃) | 안정·평야 |
| 서(西) | 금(金) | 백(白) | 결단·정제 |
| 북(北) | 수(水) | 흑(黑) | 휴식·침잠 |

출처: 한국민족문화대백과사전 「음양오행설」. 토=중앙은 사방의 중심이자 **평야·안정**을 상징.

### 1.2 지형–오행 (풍수 형기론)

풍수 형기론은 산·물의 형태를 오행으로 분류한다(오성체). 이를 지형 속성으로 일반화:

| 지형 | 오행 | 근거 |
|---|---|---|
| 산·숲 | 목 | 목성체(곧게 솟은 산), 상승하는 목기 |
| 화산·온천 | 화 | 지열·화기. 제주 화산, 온천 지열 |
| 평야·분지 | 토 | 토=중앙·안정. 평야의 안정된 기운 |
| 도심·금속 | 금 | 금=금속·결정. 밀집 도시의 정제된 기운 |
| 바다·강·계곡 | 수 | 수=흐름·침잠. 해안·하천 |

## 2. 분류 알고리즘

### 2.1 우선순위 (먼저 매칭되는 룰이 승)

```
화산  >  온천  >  산악  >  해안  >  도심  >  방위(중심좌표)
```

**왜 이 순서인가**: *자연 지형의 특이성*이 여행지 성격과 명리적 기운을 가장 강하게 규정한다.
화산·온천(지열=화)이 가장 특이, 그다음 산악(목)·해안(수). 도심(금)은 인공 특성이라 자연 지형 다음.
뚜렷한 지형 특성이 없으면 마지막으로 **방위**로 결정한다.

예) 부산 해운대구 = 광역시 자치구(도심·금)이지만 **해안(수)이 우선** → 수.
예) 부산 동래구 = 도심이지만 **온천(화)이 우선** → 화.

### 2.2 방위 default (지형 특성이 없을 때)

각 시군구의 **중심좌표(d3-geo `geoCentroid`)**를 한반도(남) 실중심 기준으로 정규화해 결정:

```js
center = (lon 127.7, lat 36.6)   // 남한 인구중심 근사
W = 1.9, H = 2.6                 // 경도/위도 반폭 정규화
ndx = (lon - 127.7) / 1.9
ndy = (lat - 36.6)  / 2.6
if  hypot(ndx, ndy) < 0.55  →  토   // 중앙 넓은 영역 = 평야·안정
elif |ndx| >= |ndy|         →  동(ndx>0)=목 / 서(ndx<0)=금
else                        →  북(ndy>0)=수 / 남(ndy<0)=화
```

- **deadzone 0.55**: 토(중앙)를 넓게 잡아 수도권·충청 등 내륙 평야가 토가 되도록. (전통상 평야=토)
- 주변부만 방위 오행을 받는다. → 경기 북부=수(북), 경북 내륙 동부=목(동), 전남 서부=금/수 등.

## 3. 룰별 상세 + 해당 시군구

### 3.1 화산 → 화 (2)
제주시, 서귀포시 — 한라산 화산섬. 화기(火)의 원형.

### 3.2 온천 → 화 (6)
지열=화. 전국 대표 온천지:
- 충남 **아산**(온양온천), **예산**(덕산온천)
- 충북 **충주**(수안보온천)
- 대전 **유성구**(유성온천)
- 부산 **동래구**(동래온천)
- 경남 **창녕**(부곡온천)

### 3.3 산악 → 목 (41)
백두대간·소백·지리·덕유·설악·태백 산악권. 산=목.
- 경기: 가평·양평·포천
- 강원(영서 산간): 태백·평창·정선·영월·홍천·횡성·인제·화천·양구
- 충북: 단양(소백)·보은(속리)·괴산·제천(월악)·영동
- 충남: 금산·청양(칠갑)
- 전북: 무주(덕유)·진안(마이)·장수·임실·남원(지리)
- 전남: 구례(지리)·곡성·담양·화순·장성·영암(월출)
- 경북: 문경·봉화·청송(주왕)·영양·영주(소백)
- 경남: 함양·산청(지리)·거창(덕유)·하동(지리)·합천(가야)
- 울산: 울주(영남알프스)

### 3.4 해안 → 수 (61)
바다·갯벌·섬. 동/서/남해안 + 도서.
- 인천: 중구·연수구·강화·옹진(섬)
- 경기: 안산단원·시흥·화성·평택·김포(서해)
- 강원: 강릉·동해·속초·삼척·고성·양양(동해)
- 충남: 보령·서산·당진·서천·태안·홍성(서해)
- 전북: 군산·부안(변산)·고창(서해)
- 전남: 목포·여수·순천(순천만)·광양·고흥·보성·장흥·강진·해남·무안·함평·영광·완도·진도·신안(다도해)
- 경북: 포항(남·북구)·영덕·울진·울릉(동해/섬)
- 경남: 통영·거제·사천·남해·고성·진해구·마산합포구(남해)
- 부산: 중구·서구·영도·사하·해운대·수영·강서·기장(해안)
- 울산: 동구·북구(동해)

### 3.5 도심 → 금 (56)
광역시·특별시 자치구(말단 "구"), 위 자연 지형에 없는 곳. 밀집 도시=금.
- 서울 25개 구 전부
- 부산: 동·부산진·남·북·금정·연제·사상구
- 대구: 중·동·서·남·북·수성·달서구
- 인천: 동·남·남동·부평·계양·서구
- 대전: 동·중·서·대덕구
- 광주: 동·서·남·북·광산구
- 울산: 중·남구

### 3.6 방위 default → 토/목/화/금/수 (84)
지형 특성이 없는 도(道)의 시·군·일반구. 중심좌표로 결정. 대부분 내륙 평야 → **토**.
- 토(중앙·평야): 세종, 수도권 다수(수원·성남·안양·용인·안산상록 등), 충북 청주권, 충남 천안권 일부, 전북 평야, 대구 달성 등
- 수(북): 경기 북부(파주·양주·동두천·연천)
- 금(서): 경기 서부(부천·고양 일부), 전남 서부 일부
- 목(동): 경북 내륙 동부(안동·경산·영천·의성 등)
- 화(남): 경남·전남 남부 일부(진주 등)

## 5. 한계 & 조정 (v1)

- **단일 오행 단순화**: 한 시군구가 산+바다를 동시에 가져도 대표 1개만. (정밀화: 복수 오행 가중 → 로드맵)
- **화(火)가 적음(15)**: 화는 화산·온천·남부에 집중. 의도된 결과(화기 지형이 실제로 희소).
- **지형 분류는 지리 지식 기반 v1**: 일부 경계 사례는 판단 차이 가능(예: 춘천=호반이라 수로 볼 여지).
- **조정 방법**: `scripts/gen-elements.mjs`의 큐레이션 세트(`HOTSPRING`/`MOUNTAIN`/`COASTAL`)에 `"시도:이름"` 한 줄 추가/삭제 후 `node scripts/gen-elements.mjs` 재생성. 동명 시군구는 시도로 자동 식별.
